# ADT-leetcode-
学习数据结构,解决相关LeetCode题  
以数据结构类型为目录,解决相关LeetCode题目来熟悉各种数据结构的应用.  
因为串,链表,队列,栈相对熟悉,先从树来练习,之后不断完善.  
提供完整的工程(工作环境Visual Studio 2017)可以运行调试查看结果,以某种数据结构为工程，不断解决LeetCode使其完善，方便以后项目中直接使用.
